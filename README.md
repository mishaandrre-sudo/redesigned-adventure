# Игры разума - Мини-игры с ИИ

Веб-приложение с коллекцией мини-игр (крестики-нолики, сапёр, шахматы, шашки, судоку, морской бой) с интеллектуальным ИИ противником.

## 🎮 Игры

### 1. **Крестики-нолики** ❌
- 3 уровня сложности (Легко, Средне, Сложно)
- Алгоритм: Minimax с alpha-beta отсечением
- Награды: Win (+6 монет, +5 рейтинг), Draw (+3 монеты, +2 рейтинга)

### 2. **Сапёр** 💣
- 3 уровня сложности (6x6, 9x9, 12x12)
- Генерация случайных полей
- Награды: Easy (+10, +8), Medium (+15, +12), Hard (+25, +20)

### 3. **Шахматы** ♟️ **[ОБНОВЛЕНО С ИИ]**
- **3 уровня сложности:**
  - Легко: случайные валидные ходы
  - Средне: Minimax depth=2 с оценкой материала
  - Сложно: Minimax depth=3 с таблицей дебютов и рандомизацией
- Полная валидация ходов для всех фигур
- ИИ ходит после игрока (задержка 500мс для UX)
- Награды: Win (+15 монет, +12 рейтинга)
- **Поражение = 0 очков (нет награды за проигрыш)**

### 4. **Шашки** ⚫ **[ОБНОВЛЕНО С ИИ]**
- **3 уровня сложности:**
  - Легко: случайные ходы с приоритетом захватов
  - Средне: Minimax depth=3, эвристика позиций
  - Сложно: Minimax depth=5 с оценкой дамок и продвижения
- Обязательные захваты фигур
- Превращение в дамки при достижении конца доски
- ИИ автоматически ходит после игрока
- Награды: Win (+20 монет, +15 рейтинга)
- **Поражение = 0 очков**

### 5. **Судоку** 🔢
- 3 уровня сложности (Легко, Средне, Сложно)
- Генерация валидных пазлов
- Решение в реальном времени
- Награды: Win (+12 монет, +10 рейтинга)

### 6. **Морской бой** 🚢 **[ОБНОВЛЕНО С ИИ]**
- Двусторонняя игра: игрок vs ИИ
- **ИИ стреляет в ответ после каждого вашего выстрела!**
- **Hunt/Target режим:**
  - Hunt: случайный поиск кораблей (паттерн в шахматном порядке)
  - Target: при попадании ИИ целит в соседние клетки
  - При добитии корабля возврат в Hunt
- Флотилия: 1x4, 2x3, 3x2, 4x1
- Очередность ходов: Игрок → ИИ (ответ)
- Награды: Win (+20 монет, +15 рейтинга)

## 💾 Локальное сохранение

Все состояние сохраняется в `localStorage` под ключом **`ef_state_v1`**:

```json
{
  "shopCoins": 100,
  "ratingCoins": 50,
  "owned": {
    "skins": ["fire", "crystal"],
    "backgrounds": ["bg_red", "bg_blue"]
  },
  "active": {
    "skin": "fire",
    "background": "bg_red"
  },
  "theme": "light",
  "profile": {
    "name": "Игрок",
    "username": "player",
    "id": "player-0001"
  }
}
```

### Правила начисления очков

- ✅ **Победа** → начисляются коины + рейтинг
- ❌ **Поражение** → **0 очков (не начисляются)**
- 🤝 **Ничья** → начисляются коины + рейтинг (если предусмотрено)

## 🔧 Технические детали

### ИИ Алгоритмы

**Minimax с Alpha-Beta отсечением:**
- Используется в шахматах и шашках
- Оптимизирует поиск, пропуская невыгодные ветви
- Позволяет увеличить глубину анализа

**Эвристики оценки позиции:**
- Шахматы: материал фигур (пешка=1, конь/слон=3, ладья=5, ферзь=9, король=1000)
- Шашки: количество фигур, дамки, продвижение по доске

**Hunt/Target в морском бое:**
- Hunt mode: случайные выстрелы в специальном паттерне (шахматный порядок)
- Target mode: активируется при попадании, целит в соседние клетки
- Приоритет: соседние клетки > случайные

### Производительность

- Шахматы (Hard): ~500мс на ход (depth=3)
- Шашки (Hard): ~500мс на ход (depth=5)
- Морской бой (ИИ ход): ~600мс (для UX паузы)
- Крестики-нолики (Hard): instant (minimax на 3x3)

## 🔗 Сохранение данных

При каждом изменении состояния автоматически:
1. Обновляется `localStorage`
2. Вызывается `app.save()`
3. Обновляется UI (header с коинами и рейтингом)

**Попытка восстановления при загрузке:**
```javascript
const saved = localStorage.getItem('ef_state_v1');
if (saved) {
  app.state = Object.assign(app.state, JSON.parse(saved));
}
```

## 📝 Инструкция по кастомизации

### Изменить награды за победу/ничью

Откройте `ef-games.js` и найдите функцию `gameEngine.award()`:

```javascript
gameEngine.award(coins, rating); // coins = монеты магазина, rating = рейтинг очков
```

### Изменить сложность ИИ

- **Шахматы:** измените `depth` в `minimax(depth, ...)` (сейчас 2 и 3)
- **Шашки:** измените `depth` в `getAIMove()` (сейчас 3 и 5)

### Добавить новую игру

1. Создайте функцию `startYourGame()` в `gameEngine`
2. Добавьте кнопку в `renderGamesScreen()`
3. Вызовите в `startGame()` при нужном id
