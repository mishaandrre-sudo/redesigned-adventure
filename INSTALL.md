### Вариант 2: Локальный HTTP сервер (рекомендуется)

**Python 3.x:**
```bash

python -m http.server 8000
# Откройте http://localhost:8000 в браузере
```

**Python 2.x:**
```bash
cd jubilant-bassoon
python -m SimpleHTTPServer 8000
# Откройте http://localhost:8000 в браузере
```

**Node.js (npx):**
```bash
cd jubilant-bassoon
npx http-server -p 8000
# Откройте http://localhost:8000 в браузере
```

**Node.js (npm package):**
```bash
npm install -g http-server
cd jubilant-bassoon
http-server -p 8000
```

**PHP (если установлен):**
```bash
cd jubilant-bassoon
php -S localhost:8000
# Откройте http://localhost:8000 в браузере
```

**VS Code Live Server:**
- Установите расширение "Live Server"
- Откройте `index.html`
- Кликните "Go Live" (правый клик → Open with Live Server)

## 🎯 ПРАВИЛА НАЧИСЛЕНИЯ ОЧКОВ

| Событие | Результат |
|---------|-----------|
| 🏆 Победа в игре | ✅ Начисляются коины + рейтинг |
| 💔 Поражение | ❌ **0 очков (НЕ начисляются)** |
| 🤝 Ничья | ✅ Начисляются коины + рейтинг |

**Пример (Морской бой):**
- Победа → +20 монет, +15 рейтинга
- Поражение → +0 монет, +0 рейтинга (ничего не начисляется)

## 🐛 УСТРАНЕНИЕ ПРОБЛЕМ

### "Странно работает ИИ в шахматах/шашках"
- Это нормально! ИИ может делать неожиданные ходы на сложных уровнях
- Используйте уровень "Легко" для более простого ИИ
- На уровне "Сложно" ИИ может делать позиционные жертвы (это тактика)

### "ИИ в морском бое слишком предсказуем"
- ИИ использует Hunt/Target режим - это нормально
- Поиск идет в "шахматном порядке" - это стратегия
- На следующих версиях можно добавить более сложный ИИ

### "Очки не сохраняются"
- Проверьте, что localStorage включен в браузере
- Если используете приватное окно/Incognito, данные удаляются при закрытии
- Очистите кэш браузера и перезагрузитесь

### "Страница медленно загружается"
- При первом открытии JS кэшируется браузером
- Последующие открытия будут быстрее
- Используйте локальный сервер (быстрее, чем с диска)

### "Некорректно отображается на мобильном"
- Откройте в портретной ориентации (если в альбомной)
- Обновите страницу (F5)
- Используйте браузер Chrome или Firefox (лучше совместимость)

## 📞 ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ

- Использует чистый ES6 (без фреймворков)
